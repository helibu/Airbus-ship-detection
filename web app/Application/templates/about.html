We are the Ship Detection Team. Welcome to our Website. I am He Li and my Teammate is Ruxin Diao. This web is an app for the Kaggle project: Airbus Ship Detection Challenge. This is the link to the Kaggle. Airbus Ship Detection Challenge

Here is the workflow of the whole model.

An ensemble model combining a ResNet50 model and a VGG16 is used to tell the ship existence. The accuracy is about 95%.
We use a U-net model with Inception V3 as backbone to locate ships on the images.
Web is built up from scratch with Flask. And the server is on AWS Elastic Beanstalk.
